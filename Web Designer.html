The more specific requirements for the Web Designer Position and information:

Technical Skills:

Graphic Design Tools:
1.) Adobe Photoshop: For creating and editing images.
2.) Adobe Illustrator: For creating vector graphics and illustrations.
3.) Sketch: An alternative to Adobe products, commonly used in web design.

Web Design Tools:
1.) HTML/CSS: Basic understanding to effectively design for the web
2.) Responsive Design: Ability to create designs that work well on both desktop and mobile devices.
3.) UI/UX Principles: Understanding of user interface (UI) and user experience (UX) design principles.

Prototyping Tools:
1.) Adobe XD
2.) Figma
3.) InVision
